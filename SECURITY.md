# Security Policy

## Supported Versions
We are currently at V3.

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability
If you find a bug that compromises user data, UPI IDs, or the settlement engine:
1. **Do not** open a public issue.
2. Email me directly at rishi.praseeth@gmail.com
3. I will acknowledge the report within 48 hours and work on a fix.
